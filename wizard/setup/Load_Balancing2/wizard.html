Use this wizard to set up basic load balancing and failover using two wireless links. The setup assumes that all traffic from the "target network" (i.e., the network "behind" this router) will be routed by this router through one of the two wireless links to the destinations, and vice versa for traffic destined to the target network.

This wizard will generate a new configuration, completely replacing the existing configuration. A reboot is required for the new configuration to take effect.

The current configuration will be replaced and a reboot is required for the new configuration to take effect. To reconnect to the router after the reboot:

  • Connect the client device (e.g., laptop) to the network port.
  • Assign a IP address within the same subnet with network.
  • Use browser to go to network's IP.
